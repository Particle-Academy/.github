# .github — Particle Academy org defaults

This repository holds the **default community-health files** for every
[Particle Academy](https://github.com/Particle-Academy) repository. GitHub
automatically applies these to any repo in the org that does not define its own:

- [`CONTRIBUTING.md`](./CONTRIBUTING.md) — how to contribute across the suite
- [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md)
- [`SECURITY.md`](./SECURITY.md) — responsible disclosure
- [`.github/ISSUE_TEMPLATE/`](./.github/ISSUE_TEMPLATE) — bug / feature issue forms
- [`.github/PULL_REQUEST_TEMPLATE.md`](./.github/PULL_REQUEST_TEMPLATE.md)
- [`profile/README.md`](./profile/README.md) — the public org profile shown on the org landing page

To override any default for a specific package, add the corresponding file to
that package's own repository.
